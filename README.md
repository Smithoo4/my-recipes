# My Recipes

A place to develop, keep, and share my recipes.

## About

This repository started as a request from one of my daughters, who asked for a
copy of all my recipes for when she grows up. It has since grown into a place
where I develop, refine, and share the dishes our family cooks.

## Recipe format

Recipes are written in [**RecipeMD**](https://recipemd.org/), a simple standard
for writing recipes as Markdown files. Specification and docs:
<https://recipemd.org/>

> **A note on Cooklang:** [Cooklang](https://cooklang.org/) is another
> plain-text recipe format worth exploring in the future. It tags ingredients,
> cookware, and timers inline, which lets tools auto-generate shopping lists and
> meal plans. It uses its own `.cook` syntax (so it does not render natively on
> GitHub), but the surrounding ecosystem is worth a look down the road.
